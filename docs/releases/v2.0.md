# Forge OS v2.0 Release Notes

发布日期：2026-06-07

## 本次重点

Forge OS 2.0 作为独立大版本从 v2.0 开始发布。这个版本把当前桌面端、移动端、Android WebView、COS 同步、周看板、每日反思、周复盘和月度 OKR 汇总为一套本地优先的个人成长系统。

## 更新内容

- 工程版本升级到 `2.0.0`，对外发布版本展示为 `v2.0`。
- Android 版本同步更新到 `versionName 2.0.0`、`versionCode 20`。
- README 补充 Forge OS 2.0 独立大版本定位。
- 桌面端周复盘保存成功后展示“周复盘已保存”摘要。
- 已保存周复盘保留“查看/编辑周复盘”入口，用户可以继续修改。
- 周复盘保存反馈支持 `aria-live="polite"` 和 `role="status"`，让保存状态可被辅助技术感知。

## 验证

- `node --test <all tests/*.test.ts>`
- `npm run lint`
- `npm run build`
- 浏览器验证：打开桌面端周复盘，填写必填项并点击保存；确认出现“周复盘已保存”摘要，并可通过“查看/编辑周复盘”重新打开表单。

## 兼容性

- 本次更新不修改周复盘、反思、任务、OKR 或同步数据结构。
- 保存逻辑仍复用现有 `saveReflection`，只增加桌面端交互反馈。
- v2.0 是 Forge OS 2.0 项目的独立首发版本；早前 1.0.x 记录保留为开发阶段变更记录。
