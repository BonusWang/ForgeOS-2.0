# Forge-OS v1.0.4 Release Notes

发布日期：2026-06-04

## 本次重点

v1.0.4 聚焦周看板浏览器端的交互细节：日历日期详情弹出框在页面滚动后仍贴近被点击日期，今日进度条改为稳定的 CSS 轨道，并扩展到卡片可用宽度的约 80%。

## 更新内容

- 修复页面滚动后点击日历日期时，详情弹出框严重错位的问题。
- 今日进度条不再使用 `░` 字符绘制未完成段，避免浏览器字体渲染造成异常块状残影。
- 今日进度条在卡片中使用约 80% 横向空间，减少空白，让完成比例更容易扫读。
- OpenSpec 正式规格新增周看板交互显示细节要求，并归档对应 change。

## 验证

- `openspec validate --all`
- `node --test tests/*.test.ts`：80 tests, 80 pass
- `npm run build`
- `npm run lint`

## 兼容性

- 本次更新仅调整周看板 UI 定位、进度条渲染和文档规格。
- 本地数据结构、COS 同步载荷和存储路径没有变更。
