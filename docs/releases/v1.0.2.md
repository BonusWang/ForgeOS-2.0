# Forge-OS v1.0.2 Release Notes

发布日期：2026-06-03

## 本次重点

v1.0.2 继续收敛桌面端工作台体验：让左侧导航更符合执行到复盘的路径，让周看板工具栏只保留周切换动作，并把模块管理弹窗从符号列表调整为更清晰的设置面板。

## 更新内容

- 左侧主导航顺序调整为 `01 周看板`、`02 周复盘`、`03 反思库`、`04 系统`。
- 周看板顶部移除重复的 `[周复盘]` 快捷按钮。
- 模块管理弹窗改为两列设置面板，显示模块区域、启用状态和启用数量。
- 启用模块使用左侧细线和状态文字表达，减少大面积强调边框。
- OpenSpec、README 和设计审计文档同步本轮 UI 决策。

## 验证

- `npm run build`
- `npm run lint`
- 浏览器验证：导航顺序正确，周看板工具栏无重复周复盘按钮，模块弹窗两列展示且无页面级横向溢出。

## 兼容性

- 不改变任务、反思、周复盘、情绪记录、模块启用或本地数据持久化的业务逻辑。
- 数据目录继续使用 `%APPDATA%\Forge\forge-data.json`。
