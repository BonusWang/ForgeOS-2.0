# Data Backup System Scope Fix / 数据备份归属系统页修复

## Why

> 中文：为什么

数据备份属于系统维护能力，不属于周看板的执行节奏。当前周看板中展示了“数据备份”模块，会让执行页面混入系统操作，也和系统页已有的数据备份入口重复。

## What Changes

> 中文：改动内容

- 周看板不再展示数据备份模块。
- 系统页新增并明确展示“数据备份”入口。
- 系统页数据备份按钮使用直观的“导出数据 / 导入数据”文案。
- 移除重复的“数据仪式 / 封存罪证 / 展开旧账”入口，统一收敛为“数据备份”。
- 移除周看板对数据备份组件的引用，避免系统维护能力出现在执行页面。

## Impact

> 中文：影响范围

- 仅调整数据备份入口所在页面。
- 不改变导入、导出、备份数据结构或持久化逻辑。
- 不影响任务、反思、模块配置等周看板业务功能。

## Out of Scope

> 中文：暂不包含

- 不重做系统页备份交互。
- 不新增自动备份能力。
- 不修改现有备份文件格式。
