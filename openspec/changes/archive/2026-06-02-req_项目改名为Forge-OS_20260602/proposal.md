# Project Rename to Forge-OS / 项目改名为 Forge-OS

## Why

> 中文：为什么

仓库名称已调整为 `Forge-OS`。项目展示名也需要同步成更明确的 `Forge-OS`，用于强调这是一个围绕个人目标确认、执行和成长复盘的生活系统，而不只是单纯的锻造隐喻。

## What Changes

> 中文：改动内容

- 页面展示品牌从 `Forge` 调整为 `Forge-OS`。
- 项目包名、构建产物名和文档仓库地址同步为 `Forge-OS` / `forge-os`。
- README 和长期规划文档补充“目标确认”的定位说明。
- 本地数据目录继续保留 `%APPDATA%\Forge`，避免改名导致用户数据路径漂移。

## Impact

> 中文：影响范围

- 影响页面标题、导航品牌、关于信息、README、CHANGELOG、构建包名和远端仓库地址。
- 不改变业务逻辑、store 数据结构、备份格式或本地数据目录。
- 不影响当前未提交的情绪趋势对齐和数据备份归属修复。

## Out of Scope

> 中文：暂不包含

- 不把数据目录从 `%APPDATA%\Forge` 改为 `%APPDATA%\Forge-OS`。
- 不新增目标确认功能页面。
- 不修改 OKR 或周复盘业务逻辑。
