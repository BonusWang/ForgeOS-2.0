# ForgeOS-2.0

Forge-OS 是一个面向个人目标确认与成长复盘的生活管理系统。它把周看板、每日反思、周复盘、月度 OKR、数据备份和 COS 同步放在同一套本地优先基板中。

当前发布版本：v2.0（工程版本 `2.0.0`）。Forge OS 2.0 从 v2.0 起作为独立大版本发布。
发布说明见 `docs/releases/v2.0.md`。

## 使用节奏

- 早上：清空收集箱，选出今日承诺。
- 白天：推进最多三件事，临时想法先收集。
- 晚上：写每日反思，留下今天的证据。
- 周末：打开周复盘，沉淀完成、卡点和下周一件调整；保存后页面会进入“周复盘已保存”摘要，可随时查看/编辑。
- 月底：查看成长证据，修正下月目标。

这套节奏是产品工作流说明，不新增独立数据模型，也不改变现有持久化、同步或 Android 私有存储契约。
